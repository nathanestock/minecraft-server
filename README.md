# 1.21.1 Modded Modpack

## Installation

### Modrinth (Recommended)

1. Download `1.21.1 Modded.mrpack`
2. Open the Modrinth App
3. Select **Import**
4. Choose the `.mrpack` file
5. Launch the instance

### Prism Launcher / MultiMC / Other Launchers

1. Download `1.21.1 Modded.zip`
2. Open your launcher (Prism, MultiMC, etc.)
3. Create a new instance
4. Choose **Import from ZIP**
5. Select the downloaded file
6. Launch the game
